Lumify

Welcome to Lumify!

Lumify extension is a simple extension that autofill luma forms.

How to use:

1. Open the extension.

2. Navigate to the Luma form you want to autofill.

3. Click on the extension icon.

4. Fill in the form.

5. Click on the add button.

6. Done!

Features:

- Autofill form

- Storage

- View

- Edit

- Delete

Notes:

1. When adding a field, make sure to add the field name as per placeholder. It's case sensitive.

2. This extension is still in development and is not yet ready for production.

3. This extension is not affiliated with Luma.
